HTML

STRUKTUR

<!DOCTYPE>

Denna deklaration specifierar vilken version av HTML som används i dokumentet.

<html>

Rotelementet i ett HTML-dokument som innehåller alla andra element.

<head>

En behållare för metadata om dokumentet, såsom titeln, länkar till stilmallar och metataggar

<title>

Definerar titeln på HTML-dokumentet, som visas i webbläsarens titelrad eller flik

<style>

Används för att bädda in CSS (Cascading Style Sheet) i ett HTML-dokument.

<link>

Specifierar externa resurser, som CSS-filer, att användas i dokumentet.

<body>

Huvudinnehållsområdet i HTML-dokumentet där text, bilder och annat innehåll visas.
